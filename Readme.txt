Descripcion de la aplicacion



En esta aplicacion tenemos un API rest el cual nos permite hacer las consulta del clima de una determinada ciudad a la pagina openweathermap.org, y regresar datos del clima de esa ciudad.

Para poder hacer eso, obtunvimos una llave de esa pagina que se utilizo en la URL de la consulta de la ciudad que se queria saber el clima, ya con eso obtenemos los valores del clima de la ciudad requerida.
